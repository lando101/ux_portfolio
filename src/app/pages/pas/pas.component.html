PAS Case Study

An enterprise grade application for managing workflows.

Key Info

Details will likely be vague to protect client info.

Overview & Problems

The original Process Automation System was deployed in 2006, using an antiquated low-code no-code solution, that was meant to manage a variety of workflow types. The technology used to build it made development fast but rigid and, as a result, the solution could not be optimized for user experience. The years following the original deployment were unresponsive to users’ feedback due to technology constraints. This led to long, frustrating, time consuming experiences for users.

In 2016, a new version of the application was approved to be built. The organization knew they wanted to create an experience that reduced the time users spent to complete tasks. However, it was now clear after years of feedback, the primary user was not obvious. Users had drastically different goals in the application. Some were approvers and others were submitters with varying expertise in the previous solution.

To design and build a successful replacement application, it had to be flexible to user needs, address radically different user goals and minimize the time spent to complete tasks.

Goals

  • Design and build a modern web application to replace the existing application
  • Create a solution that is flexible enough to provide a positive experience for users with widely differing objectives
  • Build a templatized UI that dynamically loads content and generates pages from JSON data without the need for deployments for content changes
  • Engineer an experience to help novice users identify forms that are required to be completed based on user scenario – travel, expense, security reporting, etc.
  • Develop two unique experiences (novice, expert) for users that promotes timeliness, achieved with familiar industry design patterns and contextual content visibility for convenience
  • Create an inclusive application for users with disabilities in compliance with 508 requirements
