Global4Sight

Coinfloe centralizes crypto currency data from a variety of services into an easy-to-use application designed for users new to the crypto space. Users can now explore over 250+ crypto currencies which are supplemented with additional content to help them understand the token’s use case, community, recent news and more.

Status: In Development
UI/UX Design UX Research UI Development

Overview

Coinfloe is a personal application built and designed by me. It allows users to view crypto currency information such as news, prices and more in an intuitive UI that was designed for crypto outsiders. In addition to the UI, I also built the APIs for the application which centralized a variety of crypto API services into one easy to use API. Coinfloe was built out of a passion for the crypto space but also to highlight my UX experience without NDA restrictions.

Project Scope

Brand new application

Tools

VS Code, Adobe Suite, Figma

Roles

UI/UX Designer, UI Developer, UX Researcher

Duration

6 months

Challenge

Coinfloe proved to have many challenges. The first being aggregating content from multiple providers into one intuitive UI. Crypto information is by nature complicated since it’s a combination of financials, technology, and politics. I wanted to combine these three challenging subjects into one and present it to people who were new to the space.

Secondly, Coinfloe needed more than a UX designer. It needed a front-end developer and API developer too. To make Coinfloe happen, I had to jump into all these roles.

  • Combine multiple data feeds into one application
  • Make crypto currency data digestible for crypto outsiders
  • Simplify the discovery of crypto currencies and promote exploration
